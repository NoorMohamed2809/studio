# **App Name**: HeartWise Women

## Core Features:

- Symptom Logger: Log daily health indicators (fatigue, nausea, pain, sleep, stress, heart rate, blood pressure).
- Personalized Risk Assessment: Analyze logged data to provide a risk level indicator based on user's medical history and lifestyle factors.
- Educational Content: Access articles, infographics, and videos about heart attack symptoms and preventive measures.
- Preventive Reminders: Receive push notifications for checkups, medication, and lifestyle adjustments. Tool uses an LLM tool to choose the right moment for a particular user.
- Emergency Alert: Trigger alerts to emergency contacts and local services if concerning symptom patterns are detected. Requires manual configuration of emergency contact information.
- Progress Dashboard: Track personal progress and view symptom trends over time using interactive charts.

## Style Guidelines:

- Primary color: Soft lavender (#D1B0FF) to promote calmness and trust.  This evokes an overall vibe of gentle support.
- Background color: Light gray (#F0F0F5), close to lavender, for a clean and comfortable viewing experience.
- Accent color: Muted rose (#E0AEC9) for key actions and alerts, providing a gentle contrast and highlighting important information.
- Body and headline font: 'Alegreya', a humanist serif, provides elegance and readability.
- Use minimalist, line-style icons to represent health metrics and app features.
- Subtle transitions and animations to provide feedback during interactions, enhancing user experience without being distracting.
